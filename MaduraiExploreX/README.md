# MaduraiExploreX ✈️

## Overview

MaduraiExploreX is a tourism exploration web application developed to help users discover and explore famous places in Madurai. Instead of searching separately for maps, timings, videos, and place details, users can access all information in a single platform.

The website is designed for tourists, students, and people who want to explore Madurai easily.

---

## Features

* Search famous places in Madurai
* Categories for places

  * Temples
  * Historical Places
  * Parks
  * Kids Areas
  * Museums
* Individual place pages
* Place details
* Opening and closing timings
* Google Maps integration
* Weather information
* YouTube video links
* Contact / Suggestion form
* User Login and Registration UI
* Admin Login and Dashboard
* User Dashboard
